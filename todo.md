# <PROSJEKTNAVN> — Oppgaveliste

> Aktive TODOs vises her. Fullførte oppgaver flyttes til `todo_archive.md` (opprettes ved første arkivering).

## Format

```md
### TODO [nr] — [tittel]
- [ ] Gjennomført
**Hva:** Konkret beskrivelse av oppgaven
**Hvorfor:** Kort kontekst
**Test:** Eksakte kriterier som må være oppfylt før todoen kan markeres ferdig
**Status:** ny | plan klar — venter på review | reviewet — klar for /todo-execute | under arbeid | ferdig
```

**Regler:**
- TODO-numre er sekvensielle og gjenbrukes aldri (også fra `todo_archive.md`)
- Marker `[x]` kun etter at testkriteriene er verifisert og godkjent av bruker
- Marker `[~]` for påbegynte todos med en kort note om hva som gjenstår
- Når ferdig og godkjent: flytt til `todo_archive.md` via `/todo-done`
